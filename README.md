# Hospital Management System

A professional Hospital Management System built with Spring Boot, Thymeleaf, and JPA.

## 🚀 Live Deployment

You can deploy this project to the web with a single click using Render:

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/mayakuntlathanmai-1234/hospital-management)

## Features
- Patient Registration and Management
- Doctor Management
- Appointment Scheduling
- Medical Records Tracking
- Responsive Glassmorphism UI

## Tech Stack
- **Backend**: Java, Spring Boot, Spring Data JPA
- **Frontend**: Thymeleaf, CSS3, JavaScript
- **Database**: 
    - **Local**: MySQL
    - **Cloud**: H2 (In-memory) for easy deployment

## Setup Locally
1. Clone the repository.
2. Update `src/main/resources/application.properties` with your MySQL credentials.
3. Run `mvn spring-boot:run`.
